2.CookGPT
Description

CookGPT is an AI-powered recipe generator web app that uses a locally hosted LLM (DeepSeek 7B) to provide personalized recipes based on user inputs like available ingredients, dietary preferences, and time constraints. Built with Flask and styled using HTML, CSS, and JavaScript, it features dark mode, chat history, and a dynamic typing animation to enhance user experience, making meal planning smarter and more intuitive.
